## Análisis Exploratorio de Aspectos Climatologicos Texas USA

#### Objetivo
El objetivo principal es mapear la tendencia el comportamiento del CO2.

#### Introducción
El Análisis se basa en proporcionar a los usuarios el comportamiento del CO2 con la informacion de los EEUU y visualizar su comportamiento durante años pasados

#### Desarrollo
El proceso son detallados a continuación:

1. **Conversión y Visualización Inicial:**
   En primer lugar, el dataset se importará y se gestionara el proceso EDA correspondiente.

2. **Análisis Estadístico y Correlacional:**
   La fuente de analisis se extrago de la siguiente fuente de datos
   Kaggle -> CO2 Data Cleaned.csv.

3. **Data Profiling y Reporte:**
   Finalmente, se haun utilizado herrientas graficas de forma gratutita a fin de demostrar al usuario las capacidades sin mucha inversion que les pueda brindar la herramienta.
   Se incluyo un mapa que permite al usuario el poder navegar las zonas distritales de la ciudad de Athena

#### Conclusiones
El proceso brinda una navegabilidad amigable para el usuario en base a la informacion requerida por los interesados.
